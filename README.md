# DnD-Dashboard
A simple Trello Dashboard clone in React.js and react-dnd.

<h2>Installation</h2>
Use the cmd line to install dependencies.<br>

```npm install```

Usage 
<br>
```npm run build```
<br>
```npm run dev```<br>
<br>
Add your commit notes to commit.log when committing use <br>
```git commit -F commmit.log```
